# Rustic AI NATS

This module provides a [NATS](https://nats.io/) Jetstream based implementations of [Rustic AI](https://www.rustic.ai/)'s messaging backend, key value store, etc.

## Installing

```shell
pip install rusticai-nats
```
**Note:** It depends on [rusticai-core](https://pypi.org/project/rusticai-core/)

## Building from Source

```shell
poetry install --with dev
poetry build
```
